# Ace Your IELTS 雅思衝刺班網站

這是一套多頁靜態網站，可以直接用瀏覽器開啟 `index.html` 預覽。

## 頁面

- `index.html`：首頁
- `courses.html`：完整課程資源
- `team.html`：顧問團隊
- `results.html`：學員回饋
- `enroll.html`：報名資訊與報名表單

## 結構

- `css/styles.css`：依區塊整理的共享樣式，方便後續轉 Greenshift
- `js/main.js`：手機選單、課程 tabs、靜態表單互動
- `assets/images/`：客戶素材與壓縮後頁面圖片
- `assets/generated/`：AI 生成首頁主視覺
- `reference/`：參考頁、規範與 QA 截圖留存
